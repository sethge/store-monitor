# 竞对数据采集

视频 → 提帧 → OCR → **你来分析** → 生成链接。不调任何外部AI，你自己就是AI。

## 第一步：提帧 + OCR

```bash
cd ~/.qclaw/workspace/store-monitor
python3 skills/store-diagnosis/extract_ocr.py 视频路径.mp4
```

这条命令做两件事：提取视频关键帧 + 腾讯云OCR识别文字。
输出是每张图的OCR文字（JSON格式）。

## 第二步：你来分析

读 OCR 输出，提取结构化数据，组装成以下 JSON 写入 `/tmp/competitor_data.json`：

```json
[
  {
    "店铺名称": "XX麻辣烫(XX店)",
    "平台": "美团",
    "店铺评分": 4.8,
    "营业时间": "10:00-22:00",
    "月销": "3000+",
    "实际配送费": "0元",
    "配送方式": "美团快送",
    "评价数": 1200,
    "差评数": 15,
    "差评率": 0.0125,
    "满减档位": "30-5，50-10，80-18",
    "满减档位数": 3,
    "第一档满减力度": 0.167,
    "第二档满减力度": 0.2,
    "其他活动": "折扣商品5折起; 新人立减12",
    "热销菜": [
      {"名称": "招牌麻辣烫", "月销": "800+", "实际价格": 25.8, "折扣力度": 0}
    ]
  }
]
```

规则：
- 热销菜取月销Top10，去掉蘸料/小料/饮料/米饭/配菜/单点不送
- 去掉"福利放送"分类下的菜品（价格极低如¥0-5，标"X份起购"）
- 满减力度 = 减免额 ÷ 门槛
- 折扣力度 = 有划线价时 折扣价÷原价，没有则为0
- 看不到的写 "未获取"

## 第三步：生成链接

```bash
python3 skills/store-diagnosis/deploy.py --data /tmp/competitor_data.json
```

输出一个本地链接，发给运营。

## 发链接给运营

```
数据提完了，打开这个链接查看竞对报告：
<链接>

• 数据可以直接点击修改
• 底部分析区需要填写
• 输入你的店铺名，点下载导出Excel
```

## 红线

1. **必须走完三步到生成链接。** 不能只输出文本总结就停。
2. **第二步你自己分析。** 不要调任何外部API，你就是AI。
3. **环境问题跑 setup.sh。** 不甩给运营。
