# 运营规则

_从经验中沉淀的规则。Seth审核后写在这里，agent每次执行任务前读这个文件并遵守。_

## 通知过滤规则

- 发票类通知：不报
- 活动到期提醒（平台自带）：不报，但自己的活动即将到期要报
- 标题含【】的平台广告：不报
- 招商/上线/升级/覆盖：不报

## 预警规则

- 推广余额不足1天才预警，提前太多运营嫌烦
- 活动到期提前3天开始提醒
- 差评只报近3天的，更早的不翻旧账

## 沟通规则

- 全部正常时一句话带过，不要长篇大论
- 有问题时说重点，不要列所有检查项
- 预警无新增不说话
